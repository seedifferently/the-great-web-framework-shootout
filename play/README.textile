h1. Play 2 code for the Great Web Framework Shootout

h2. How to run this sample?

 * Build "Play 2":https://github.com/playframework/Play20 from master sources
 * Using a terminal, go in the @the-great-web-framework-shootout/play/@ directory
 * Run the @play run@ command

h2. How to benchmark this sample?

 * @$ play clean compile stage@
 * @$ target/start &@
 * @$ ab -n 1000000 -k -c 300 localhost:9000/hello@
 * @$ ab -n 1000000 -k -c 300 localhost:9000/hellos@
 * @$ ab -n 1000000 -k -c 300 localhost:9000/hello-db@
